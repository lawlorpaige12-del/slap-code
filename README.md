# slAP Study & Productivity Ecosystem

This repository contains a starter Vite + React + TypeScript app for slAP.

## Setup

1. Install Node.js and npm or another compatible package manager.
2. Run `npm install`.
3. Run `npm run dev` to start the development server.

## Project structure

- `src/` — app source code
- `src/pages/` — page views for Home, Study Planner, Pomodoro, Study Tools, and Contact Us
- `src/components/` — reusable UI components
- `assets/images/` — image asset folder for custom icons and illustrations
- `styles.css` — global styling and palette definitions

## Notes

- The app uses the slAP color palette and Federo-inspired typography.
- The `assets/images/` folder contains a `.gitkeep` to preserve the directory and a README for asset instructions.
- No generated images are included in the repository.
